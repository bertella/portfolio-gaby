# Portfolio Profesional — Gabriel

### Operations & Data Analyst | Real Estate Professional

Este es mi portfolio personal desarrollado con **React** y **Tailwind CSS**. Aquí centralizo mi experiencia en análisis de datos, automatización de procesos y desarrollo de soluciones Low-Code.

## 🚀 Proyectos Destacados
* **Market Intelligence:** Análisis espacial con Tableau para cadenas de café.
* **Retail Performance Engine:** Dashboards de rentabilidad regional con SQL.
* **Inventory Ecosystem:** Automatización de stock con AppSheet y AppScript.

## 🛠️ Stack Técnico
* **Data:** SQL, Tableau, Power BI, Looker Studio.
* **Dev/Automation:** React, AppSheet, Google AppScript, Supabase.

## 📈 Impacto
Mi enfoque combina la inteligencia de datos con la eficiencia operativa, transformando datos fragmentados en herramientas que disparan acciones de negocio.
